PAC-FONT INSTRUCTIONS
====================

To use the authentic Pac-Font in this game:

1. Download the Pac-Font from one of these sources:
   - https://www.dafont.com/pacfont.font
   - https://famfonts.com/pac-man/
   - Search for "Pac-Font" by Fontalicious

2. Extract the font file (usually named "PacFont.ttf" or "PAC-FONT.TTF")

3. Place the font file in this fonts/ directory

4. The game will automatically use the font for the "PACMAN" title

Note: The Pac-Font features outlined capital letters and solid lowercase letters,
making it perfect for creating the authentic Pac-Man logo look.
